# 🔹 Arrays & Hashing

## 🧩 Problem 1: Contains Duplicate (**LeetCode 217**)

### Question

Check if an array contains any duplicate elements.

---

### ✅ Brute Force

**Idea:** Compare every pair of elements
**TC:** O(n²), **SC:** O(1)
**Interview Line:** Compare all pairs to detect duplicates

---

### ✅ Sorting

**Idea:** Sort array and check adjacent elements
**TC:** O(n log n), **SC:** O(1)
**Interview Line:** Sort and compare adjacent elements

---

### ✅ HashSet (Best)

**Idea:** Store elements and check if already exists
**TC:** O(n), **SC:** O(n)
**Interview Line:** Use HashSet to detect duplicates in linear time

---

### 🔥 Final

**Best → HashSet (O(n))**
